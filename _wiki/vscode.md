Configuration
-------------

- Disable telemetry

```
  "telemetry.enableTelemetry": false,
```

- Don't ask for syncing with git

```
  "git.confirmSync": false
```

- Indent Python files with 2 spaces instead of 4

```
  "python.formatting.autopep8Args": [
    "--indent-size=2"
  ]
```
